Desarrollo de la Lista de Compras – Odoo 17 EE

Este documento describe las funcionalidades del módulo de Lista de Compras, diseñado para optimizar el proceso de pedidos de compra en Odoo mediante la introducción de funcionalidades avanzadas y características personalizadas.

Funcionalidades del Módulo

  1. Se añade un nuevo menú en Compras: Se crea un menú denominado "Lista de Compras" dentro del módulo de Compras.
  2. Funcionalidad del botón para generar la lista de compra: Se incluye un botón denominado "Generar Lista de Compra". Al presionar este botón, se recuperan automáticamente todos los productos encontrados en las órdenes de compra en estado de "RFQ" o "Cotización de compra".
  3. Visualización y filtrado de la lista: Se presenta los productos en una lista tipo checklist, mostrando las columnas "Nombre del Producto", un "Checkbox" para marcarlos como comprados y una columna opcional Imagen. La lista permite filtrar por proveedor para facilitar la gestión de las compras.
  4. Confirmación automática de pedidos y entradas: Se añade un botón llamado "Confirmar ordenes de compra y Picking " que, al ser presionado, confirma automáticamente los pedidos de venta y sus entradas asociadas con la lista de compras.
